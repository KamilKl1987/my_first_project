# Kalkulator CLI (Python)

Prosty kalkulator działający w terminalu, napisany w Pyhtonie.
Projekt szkoleniowy wykonany w celu nauki języka Python oraz dobrych praktyk programistycznych.

## Funkcjonalności 
- menu tekstowe w terminalu 
- dodawanie
- odejmowanie
- mnożenie 
- dzielenie (z obsługą dzielenia przez zero)
- możliwość wykonania wielu operacji w jednej sesji 
- bezpieczne zakończenie prgramu 


## Technologia
- Python 3.x
- uruchmiany w środowisku terminalowym (CLI)

## Uruchamianie projektu 

1. Sklonuj repozytorium 
2. (opcjonalnie) utwórz i aktywuj virtualenv
3. Uruchom program:

```bash
python src/main.py
